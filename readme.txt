INSTALLATION DE L'APPLICATION POUR LES DISTRIBUTIONS UBUNTU:

N.B.: Verifiez le contenu de install.sh avant de lancer l'installation.

1)Ouvrez un terminal.
2)Placez vous dans le repertoire QGLLearning

3)Tapez sudo ./install.sh
ou
3')Tapez sudo ./installKDev.sh si vous souhaiter obtenir un dossier integrable directement dans l'EDI KDevelop.

4)Tapez votre mot de passe (rien ne se passe à l'ecran , c'est normal)
5)Laissez le travail se faire et suivez les instructions
6)./QGLLearning pour lancer l'application
